Fluxo Detalhado dos Módulos

1. Módulo Operacional - Capacidade Produtiva

Input: Número de turnos (1-3)
Validar: 1 ≤ turnos ≤ 3
Calcular: 1.666 unidades × turnos
Projeções: Dia, Mês (×30), Ano (×12)
Análise: Ociosidade e % uso
Exibir: Relatório formatado

2. Módulo Estoque - Entrada de Produtos

Input: Nome, Quantidade, Preço
Validar: Nome ≥ 3 chars, Qtd > 0, Preço > 0
Buscar produto no banco (por nome)
Produto existe?
Atualizar quantidade
Criar novo produto
Salvar no Banco (commit)
Exibir: Confirmação

3. Módulo Estoque - Saída de Produtos

Listar todos os produtos do banco
Exibir: Tabela com ID, Nome, Qtd, Preço
Input: ID do produto, Quantidade
Buscar produto por ID
Estoque suficiente?
ERRO: Estoque insuficiente
Subtrair quantidade
Atualizar banco (commit)
Calcular: Total da venda
Exibir: Comprovante

4. Módulo Financeiro - Custos e Lucros

Input: Água, Luz, Impostos, Salários
Validar: Valores ≥ 0
Somar: Custo Total
Calcular: Custo por unidade (÷1000)
Aplicar: Margem de 50%
Calcular: Preço de Venda
Projeções: Receita e Lucro (mensal × 12)
Indicadores: ROI, Ponto de Equilíbrio
Exibir: Relatório Financeiro Completo

5. Módulo RH - Folha de Pagamento

Input: Número de funcionários
LOOP: Para cada funcionário
Input: Nome, Cargo, Horas Extras
Buscar: Valor/hora do cargo
Calcular: Salário Bruto (160h × valor)
Tem HE?
Adicionar: HE × valor × 2
Calcular: INSS (tabela progressiva)
Base IR: Bruto - INSS
Calcular: IR (tabela progressiva)
Líquido: Bruto - INSS - IR
Adicionar à lista de funcionários
Ordenar: Por salário (maior → menor)
Totalizar: Bruto, INSS, IR, Líquido
Exibir: Relatório da Folha
